1. HTML Programming Assignments

    Assignment
  • 1: Basic HTML Structure Concept: Basic HTML tags, structure. Task: Create a simple HTML page with a heading, paragraph, and a link to another webpage. Assignment
  • 2: Create a Simple Table Concept: Tables in HTML. Task: Write an HTML program to create a simple table with at least 3 rows and 3 columns. Assignment
  • 3: Create a Contact Form Concept: Forms in HTML. Task: Create a form with input fields (name, email, and message), and a submit button. Assignment
  • 4: Image Gallery Concept: Images and links in HTML. Task: Create an image gallery with at least 4 images that are linked to other pages. Assignment
  • 5: Create a List of Items Concept: Unordered and ordered lists. Task: Create both an ordered and unordered list of 5 items each, with links for each list item.

CSS Programming Assignments

Assignment
  • 1: Basic CSS Styling Concept: Basic CSS styling (colors, fonts, background). Task: Style a simple HTML page (from the HTML Assignment 1) with CSS. Set a background color, text color, font family, and padding.
  • 2: Layout with Flexbox Concept: Flexbox for layout. Task: Create a layout using flexbox with two columns. One column should take 70% of the width, and the other should take 30%. Assignment
  • 3: Responsive Navigation Bar Concept: Media queries, navigation bars. Task: Create a responsive navigation bar that changes from horizontal to vertical when the screen width is less than 768px. Assignment
  • 4: Hover Effects Concept: CSS pseudo-classes. Task: Style a button to change color when hovered over using the :hover pseudo-class. Assignment
  • 5: CSS Grid Layout Concept: CSS grid for layout. Task: Create a 3x3 grid layout with 9 items. Each item should have a border and some padding.
